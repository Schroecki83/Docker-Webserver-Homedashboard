# Home Dashboard

Phase 1 scaffold for a Raspberry Pi home dashboard integrating Fronius, Shelly LAN, and Luxtronic.

## Start

1. Copy `.env.example` to `.env`.
2. Run `npm install`.
3. Run `npm run dev`.

## Current Scope

- Next.js app shell
- Environment validation
- Health endpoint
- Core telemetry types
- Fronius client and mapper
- Initial tests
